# octo

Задание: Нужно сверстать страничку по макету из Figma, сделать адаптив.
Будет плюсом, если ты сможешь реализовать слайдер с карточками (четвертая секция).
Для выполнения задания используй свой стек, к которому ты привык.
В результате нужно скинуть билд проекта и его исходники на почту.
Ссылка на макет:

Несколько комментариев по выполненному тестовому заданию:
1. нет адаптивности шрифтов, отступов из макета на разрешении от 992 до 1200
2. нет уменьшения количества карточек на адаптиве
3. адаптивность solutions блока не соответствуют макету по размерам колонок
4. некоторые картинки не совпадают по кадрированию с макетом
5. для карточек во features и слайдере используются разные классы/блоки
